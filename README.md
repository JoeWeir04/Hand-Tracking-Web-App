# Hand Tracking Web App
Mobile system using hand tracking input prototype. Utilizing Mediapipe hand tracking JavaScript Library. Multimodal feedback upon input into the system.

## How to run
Open with live server or in terminal:
Run server 
```bash
python -m http.server 8000
```
Then visit: http://localhost:8000 in any web browser.
